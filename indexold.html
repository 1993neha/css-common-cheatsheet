CSS Box Model

Property Definition
Content The content area of an element. The area of the box that is outlined by the padding, border, and margin.
Padding The space between the content area and the border. The size of the padding is defined by the padding-top, padding-right, padding-bottom, and padding-left properties.
Border The area between the padding and the margin. The size of the border is defined by the border-top, border-right, border-bottom, and border-left properties.
Margin The space around the outside of the border. The size of the margin is defined by the margin-top, margin-right, margin-bottom, and margin-left properties.

Position Property

Position Description CSS Example
Static The default value. Elements render in the normal flow of the document. .example-static { position: static; }
Relative The element is positioned relative to its normal position, and can be offset using the top, right, bottom, and left properties. .example-relative { position: relative; top: 10px; left: 10px; }
Absolute The element is positioned relative to the nearest positioned ancestor (or the document body, if no positioned ancestor exists). The element is removed from the normal document flow. .example-absolute { position: absolute; top: 20px; left: 20px; }
Sticky The element is positioned relative to its normal position until a specified threshold is crossed (e.g., when the element reaches the top of the viewport), at which point it is treated as fixed. .example-sticky { position: sticky; top: 30px; }
Fixed The element is positioned relative to the viewport, and remains fixed in place as the page is scrolled. .example-fixed { position: fixed; top: 40px; left: 40px; }

Display Property

Property Description CSS Example
block The element will display as a block-level element, taking up the full width of its container. .container1 { display: block; }
inline The element will display as an inline-level element, only taking up as much space as necessary. .container2 { display: inline; }
inline-block The element will display as an inline-level element, but with block-level properties like width and height. .container3 { display: inline-block; }

Colors Property

Property Description CSS Example
Name Keywords Name keywords can be used to set color property values for elements in CSS. h1 { color: red; }
rgb() rgb() should be supplied with three values representing red, green, and blue. These values range can from 0 to 255. .green { color: rgb(0, 255, 0); }
hsl() This syntax contains three values: hue (the color value itself), saturation (intensity), and lightness. .light-blue { background-color: hsl(200, 70%, 50%); }
Hexadecimal Colors Hexadecimal digits can represent sixteen different values using 0-9 and a-f. Hexadecimal colors are composed of 6 characters each group of two represents a value between 0 and 255 for red, green, or blue. .red { color: #ff0000; } .short-blue { color: #00f; }
Alpha Values Alpha values determine the transparency of colors in CSS. Alpha values can be set for both RGB and HSL colors by using rgba() and hsla() and providing a fourth value representing alpha. .midground { background-color: rgba(0, 255, 0, 0.5); } .foreground { background-color: hsla(34, 100%, 50%, 0.1); } .transparent { color: transparent; }
